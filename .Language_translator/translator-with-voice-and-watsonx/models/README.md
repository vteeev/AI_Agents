# README for the models used in the translator with voice and WatsonX application

This directory contains the models utilized in the translator application that integrates voice capabilities and WatsonX services.

## Structure

- **stt/**: Contains models and resources for speech-to-text functionality.
- **tts/**: Contains models and resources for text-to-speech functionality.

## Usage

To use the models, ensure that the necessary dependencies are installed and configured properly. Refer to the documentation in the respective directories for detailed instructions on how to implement and utilize the models effectively.
