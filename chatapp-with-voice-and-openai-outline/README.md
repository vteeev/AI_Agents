# Chat Application with Voice and OpenAI

This project is a chat application that integrates voice capabilities and utilizes OpenAI's API for enhanced conversational experiences. 

## Features

- Voice input and output for a seamless user experience.
- Integration with OpenAI's language models for intelligent responses.
- User-friendly interface for easy interaction.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd chatapp-with-voice-and-openai-outline
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

4. Set up environment variables in the `.env` file.

## Usage

To start the application, run the following command:
```
python server.py
```

Access the application in your web browser at `http://localhost:8000`.
