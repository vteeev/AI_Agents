# Models Documentation

This directory contains the models used in the application, specifically for speech-to-text (STT) and text-to-speech (TTS) functionalities. 

## Structure

- **stt/**: Contains the models and resources related to speech-to-text processing.
- **tts/**: Contains the models and resources related to text-to-speech processing.

## Usage

To utilize the models, ensure that the necessary dependencies are installed and the environment is properly configured. Refer to the respective model directories for specific instructions on usage and setup.
